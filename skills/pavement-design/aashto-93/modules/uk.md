### M.1.2 Institutional Context

The UK does not use AASHTO 93 for pavement design. UK pavement design follows DMRB CD 227, which uses a different analytical framework based on foundation classes and standard designs.

### M.1.3 Standard Philosophy

UK uses an analytical approach with standardised designs verified by mechanistic analysis. The concept of structural number is not used. Instead, designs are selected based on foundation class (subgrade stiffness) and design traffic in msa.
