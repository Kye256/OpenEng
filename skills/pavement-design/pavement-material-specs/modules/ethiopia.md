### M.1.2 Institutional Context

ERA's material specification system defines quality requirements for each pavement layer. Material codes (GB1, GB2, GB3, GS, GC for unbound; CB1, CB2 for cemented; AC for asphalt) are used throughout the design catalogue to specify what material type is required at each layer. The design catalogue charts (A1-D) prescribe material codes for each pavement structure based on surfacing type, base type, traffic class, and subgrade class.

> Cross-reference: See Skill #13 Ethiopia module (M.1.2) for full institutional context.

### M.1.3 Standard Philosophy

ERA uses a material code system where each code represents a defined set of grading, plasticity, strength, and durability requirements. Materials are tested against these requirements using AASHTO/ASTM methods and must meet all criteria for the assigned code. The system is derived from TRL ORN 31 and South African practice. ERA material codes are not directly interchangeable with Uganda MoWT or Kenya RDM material codes, though the underlying properties are comparable.

### M.2.4 Cross-Country Comparison: Material Classification Systems

| ERA (Ethiopia) | MoWT (Uganda) Equivalent | RDM (Kenya) Equivalent | Notes |
|---------------|-------------------------|------------------------|-------|
| GB1 (Crushed Stone) | G80 (Crushed Stone) | GCS (Graded Crushed Stone) | Similar concept, different specification details |
| GB2 (Natural Gravel) | G60 (Natural Gravel) | G80 (Gravel 80% CBR) | ERA and Uganda require 80% CBR; Kenya G80 is similar |
| GB3 (Natural Gravel, lower) | G45 (Natural Gravel) | G45 (Gravel 45% CBR) | ERA GB3 still requires 80% CBR; differs from Uganda/Kenya G45 |
| GS (Granular Subbase) | G30/G25 (Subbase) | G30 (Gravel 30% CBR) | ERA 30% CBR at 95% compaction; broadly similar |
| GC (Capping) | G15 / Selected Fill | G20 (Gravel 20% CBR) | ERA 15% CBR at 93% compaction |
| CB1/CB2 (Cemented Base) | C3/C4 (Cemented) | HIG/HMS/HMB (Hydraulic) | Kenya uses different stabilised material classification |

**Key difference:** ERA GB3 requires 80% CBR (same as GB1 and GB2) but with wider grading and plasticity tolerances. This differs from Uganda's G45 (which requires only 45% CBR) and Kenya's equivalent grades.

## M.3 Country-Specific Reasoning

### M.3.1 Decision Logic

**Step 1: Determine pavement structure from design catalogue**
- Surfacing type (surface treatment, thin AC, thick AC) + traffic class (T1-T8) + subgrade class (S1-S5) determines the chart group (A1-D).
> Cross-reference: See Skill #10 Ethiopia module for subgrade class.
> Forward reference: See Skill #21 Ethiopia module (when created) for design catalogue charts.

**Step 2: Identify material code for each layer**
- The design catalogue chart specifies the material code for each pavement layer (base, subbase, capping).
- Example: Chart A1 for surface treatment with granular base may specify GB1/GB2 base + GS subbase + GC capping.

**Step 3: Verify available materials against specification**
- Test locally available materials against the required material code specification (Tables 6-1 to 6-7).
- If materials meet or exceed requirements: proceed.
- If materials fall short: consider stabilisation (Chapter 7), material import, or design modification.

**Step 4: Check climate-specific requirements**
- GB2/GB3 PI limits differ between wet and dry climates (Table E.24.3).
- In wet climate areas: use lower PI limits (more stringent).
- In dry climate areas: higher PI limits are permitted.

### M.3.2 Professional Judgment

- Ethiopian laterite gravels commonly meet GB2/GB3 grading and CBR requirements but may fail PI limits. Consider lime modification to reduce PI.
- Crushed volcanic rock (basalt) is widely available in Ethiopia and typically meets GB1 requirements. Verify ACV and TFV as some vesicular basalt can be weak.
- In remote areas where GB1 crushed stone is not available, GB2 or GB3 natural gravel bases may be the only practical option even for higher traffic roads.
- ERA's uniform 80% CBR requirement for GB1/GB2/GB3 makes the grading and PI limits the differentiating criteria between these codes.

### M.3.3 Common Errors

1. **Assuming ERA GB3 = Uganda G45 or Kenya G45.** ERA GB3 requires 80% CBR (same as GB1/GB2), just with wider grading and PI tolerances. Uganda G45 and Kenya G45 require only 45% CBR.
2. **Applying Uganda or Kenya material codes to Ethiopian projects.** ERA uses GB1/GB2/GB3/GS/GC. Never use G-codes (Uganda) or Kenya material codes for ERA projects.
3. **Ignoring wet/dry climate distinction for PI limits.** ERA GB2 and GB3 have different PI limits for wet and dry climates (Table 6-6). Using dry climate limits in a wet area is non-conservative.
4. **Using penetration grade binder where performance grade is specified.** ERA standard practice uses penetration grade (60/70, 80/100). PG binders are not standard ERA specification.

## M.5 Country-Specific Validation

### M.5.1 Range Checks

| Parameter | Valid Range | Flag If Outside |
|-----------|------------|-----------------|
| Base CBR (soaked) | 60-150% | CBR <80% fails GB code; CBR >120% verify test |
| Subbase CBR (soaked) | 20-80% | CBR <30% fails GS code |
| Capping CBR (soaked) | 10-40% | CBR <15% fails GC code |
| PI for base (wet climate) | 0-10 | PI >6 fails GB1; PI >10 fails GB2/GB3 wet |
| LA Abrasion | 10-60% | LA >45% fails GB1; LA >51% fails GB2/GB3 |
| UCS for stabilised base | 0.5-8.0 MPa | UCS <1.5 MPa below CB2; UCS >6.0 MPa above CB1 (risk of shrinkage cracking) |

### M.5.2 Standards Compliance

- Material codes shall use ERA designation (GB1, GB2, GB3, GS, GC, CB1, CB2).
- All CBR tests at Modified AASHTO compaction (T-180) for base and subbase materials.
- Aggregate tests shall reference AASHTO T-96 (LA Abrasion), BS 812 (ACV, TFV, Flakiness).
- Climate zone (wet/dry) shall be stated for PI limit selection.

### M.5.3 Departures Process

Any departure from ERA material specifications (e.g., accepting material that fails one parameter) requires written approval through the regional directorate to the Quality Assurance, Road Inspection and Safety Directorate. Common departure requests include relaxed PI limits for natural gravels in areas with limited material sources.

## M.6 Country-Specific Escalation

| Trigger | Reason | Action |
|---------|--------|--------|
| No locally available material meets GB code | Cost and programme impact of material import | Investigate stabilisation options (CB1/CB2); escalate to senior pavement engineer |
| PI exceeds specification by >50% | Material fundamentally unsuitable without treatment | Consider lime/cement modification; reject material if treatment not viable; escalate |
| LA Abrasion >51% for base material | Aggregate too soft for base application | Reject material; investigate alternative sources; escalate |
| Stabilised material UCS >6.0 MPa | Risk of drying shrinkage cracking | Reduce cement content; consider lime as alternative binder; escalate |
| Binder supply limited | Only one pen grade available locally | Assess suitability for climate; escalate for specification modification |

## M.7 Worked Example

### Material Code Selection for Mekelle - Adigrat Road

**Project:** Mekelle to Adigrat road upgrade, DC3 trunk road, traffic class T5, subgrade class S3.

**Design catalogue output (Chart B):** AC surfacing (<50 mm) with granular base.
- Surface: 40 mm AC wearing course
- Base: 150 mm GB1 or GB2
- Subbase: 175 mm GS
- Capping: 150 mm GC (required for S3 subgrade)

**Available material: Quarried basalt from Mekelle quarry**

Test results:
- CBR at 98% Mod AASHTO: 95% (soaked)
- Grading: Meets Table 6-2 envelope for GB1
- PI: 3 (non-plastic fines)
- LL: 22
- LA Abrasion: 28%
- ACV: 22%
- TFV: 145 kN (dry), 98 kN (wet)
- Flakiness: 25%
- Soundness: 8%

**Assessment against GB1 requirements (Table E.24.2):**
- CBR 95% >= 80%: PASS
- PI 3 <= 6: PASS
- LL 22 <= 25: PASS
- LA 28% <= 45%: PASS
- ACV 22% <= 29%: PASS
- TFV 145/98 kN >= 110/75 kN: PASS
- Flakiness 25% <= 35%: PASS
- Soundness 8% <= 12%: PASS

**Result:** Mekelle quarry basalt qualifies as **GB1 (Graded Crushed Stone)**. Material meets all ERA specification requirements for the highest quality unbound base.

> **All outputs are preliminary/screening level.** Verify material specification against full laboratory test programme and ERA Pavement Vol I Chapter 6.
