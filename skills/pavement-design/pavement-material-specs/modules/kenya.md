### M.1.2 Institutional Context

Kenya's pavement material specifications are defined in RDM 3.3 Chapter 8 and the Materials Specification Charts (GM1-GM13 for granular, HM1-HM5 for hydraulically improved/modified/bound, BM1-BM3 for bitumen stabilised, BB1-BB2 for bituminous base, SU1-SU11 for surfacing, PB1-PB2 for paving blocks, CP1-CP4 for concrete pavement). These charts are the definitive source for material requirements.

- **KeNHA:** Specifies materials for Class A and B trunk roads -- typically requires GCS or G80 base.
- **KeRRA:** Specifies materials for rural roads (C, D, E) -- may use G30, G45, or HIG materials.
- **KURA:** Specifies materials for urban roads -- full range including asphalt and concrete.
- **County Governments:** Manage county roads; often use natural gravel (G20 wearing course) for unpaved roads.

> Cross-reference: See Skill #13 Kenya module (M.1.2) for full institutional context.

### M.1.3 Standard Philosophy

Kenya uses a distinct material classification system where materials are coded by type and minimum strength. The system is independent from Uganda's BS-based G-code/C-code system. Specification requirements are minimum values defined in the Materials Specification Charts. Each chart specifies the material requirements, application, and construction procedures for a single material code.

Climate resilience measures in the 2025 RDM have made plasticity specifications more stringent for granular and hydraulically modified materials to minimise moisture susceptibility (RDM 3.3 Section 3.4).

### M.2.2 Country-Specific Formulas

No formulas -- specification lookup from material charts.

**Plasticity Modulus (PM):** PM = PI x (% passing 0.425 mm sieve). Used as a specification parameter for several material classes (G14, G20, G23, G25, G30, G45, G80).

### M.2.3 Classification System

Kenya uses a material coding system where:
- **G + number** = natural/blended granular material, number = minimum soaked CBR (e.g., G30 = min CBR 30%)
- **GCS + letter** = Graded Crushed Stone, letter = aggregate quality class (F = weakest, A = strongest)
- **HIG + number** = Hydraulically Improved Granular, number = minimum CBR after 7-day cure
- **HMS + number** = Hydraulically Modified Stone, number = minimum UCS in kPa (cylinder strength)
- **HBS + number** = Hydraulically Bound Stone, number = minimum UCS in kPa
- **BSM + number** = Bitumen Stabilised Material, number = minimum soaked ITS in kPa
- **BB + number** = Bituminous Base (asphalt base / dense bitumen macadam), chart number indicates specification level (BB1 = standard DBM, BB2 = high modulus)
- **SU + number** = Surfacing, chart number indicates surfacing type: SU1-SU4 = asphalt concrete by nominal aggregate size, SU5-SU11 = surface dressing types (SSD, DSD, Cape seal, Otta seal, sand seal, slurry/micro-surfacing)
- **AC + number** = Asphalt Concrete, number = nominal maximum aggregate size in mm (AC 20, AC 14, AC 10)
- **HPS** = Hand Packed Stone
- **MAC** = Macadam (dry-bound or wet-bound)

This is INDEPENDENT from Uganda's system. Do NOT map Kenya G30 to Uganda G30 -- they have different specifications and test conditions.

## M.3 Country-Specific Reasoning

### M.3.1 Decision Logic (Kenya Branch)

```
IF Kenya project:
  1. Determine traffic class from RDM 3.4 (TC0.025 to TC150+)
  2. Determine subgrade class from RDM 3.3 Table 4.1 (S1-S6)
  3. Select pavement structure from RDM 3.4 standard structure types
  4. For each layer in the structure:
     a. Identify material code from structure sheet (e.g., G45, GCS-D, HIG100)
     b. Look up specification in corresponding Chart (GM1-GM13, HM1-HM5, etc.)
     c. Verify available material meets ALL requirements:
        - CBR or UCS at specified density and soak condition
        - Plasticity limits (PI, LL, PM) -- note wet/dry area differentiation
        - Grading envelope
        - Aggregate quality (LAA, ACV) where applicable
        - Swell limit
        - Construction requirements (layer thickness, compaction)
  5. For surfacing selection:
     a. IF structure type specifies AC surfacing:
        - Select AC type by traffic class: AC 10 for TC1-TC3, AC 14 for TC3-TC30, AC 20 for TC30+
        - Select binder grade by climate zone (60/70 inland, 80/100 coastal, PMB for TC30+)
        - Verify aggregate meets LAA Max 30%, ACV Max 25% per Charts SU1-SU4
     b. IF structure type specifies surface dressing:
        - Select type by traffic class and aggregate availability:
          * SSD for TC0.025-TC1 (new) or resealing
          * DSD for TC0.5-TC10 (new) or resealing TC3-TC10
          * Cape seal for TC3-TC17 (improved waterproofing)
          * Otta seal for TC0.025-TC3 (lower aggregate quality areas)
        - Verify aggregate ACV and flakiness per Charts SU5-SU11
     c. IF structure type specifies bituminous base:
        - BB1 (DBM) for TC17-TC50
        - BB2 (high modulus DBM) for TC50+
        - Verify aggregate meets Charts BB1-BB2 requirements
  6. If material does not meet specification:
     a. Check if treatment (hydraulic improvement) can upgrade to required code
     b. Check if alternative material class is available for the traffic/subgrade combination
  7. DO NOT use Uganda material codes (G80-Uganda, C1, C2, C3) for Kenya projects
```

### M.3.2 Professional Judgment (Kenya-Specific)

**Kenya material availability by region:**
- **Central/Western Kenya:** Good lateritic gravels (G30-G45 typical), quartzitic gravels (G45-G80), crushed stone from granite/gneiss quarries
- **Rift Valley:** Volcanic materials -- tuffs, scoria (variable quality), some excellent basalt for crushing (GCS-A/B)
- **Coastal Region:** Coral gravels (G25-G45), limited crushed stone, sand abundant
- **Northern Kenya (Arid):** Calcareous gravels, calcrete (variable), limited water for hydraulic treatment
- **Lake Victoria Basin:** Sandy clays on volcanics, lateritic gravels of variable quality

**The G80 60% alternative (Chart GM10 note):** In areas with annual rainfall < 500 mm, if permission is obtained from the Chief Engineer (Materials), material of CBR 60% may be used as an alternative to G80 base for TC1. This is a departure requiring formal approval.

**Climate differentiation:** Several material charts (G20, G25) have different PI limits for wet areas vs dry areas. In wet areas (rainfall > 500 mm), lower PI limits apply to reduce moisture susceptibility.

**Bituminous binder grade by climate zone:**
- **Coastal Kenya (Mombasa, Malindi, Lamu):** Mean annual temperature > 27C; use softer 80/100 pen grade to prevent cracking from thermal stress; asphalt temperatures are high, so softer binder deforms less during placement.
- **Highland Kenya (Nairobi, Eldoret, Nyeri > 1500m):** Cooler climate; 60/70 pen standard, 40/50 pen for heavily loaded areas.
- **Rift Valley / Northern Kenya (hot arid):** High pavement temperatures; 60/70 pen standard, PMB for TC17+.
- **PMB requirement:** Polymer modified binder is required for TC30+ traffic classes to resist rutting under channelised heavy loading. This is a Standard Specification requirement, not a design option.

### M.3.3 Common Errors (Kenya-Specific)

1. **Mapping Kenya G-codes to Uganda G-codes as equivalents.** Kenya G80 (Chart GM10) has different specifications from Uganda G80. Test conditions, aggregate quality requirements, and grading envelopes differ.
2. **Ignoring wet/dry area PI differentiation.** Charts GM5 (G20), GM7 (G25) specify different PI limits for wet and dry areas. Using dry-area limits in a wet area violates the specification.
3. **Specifying G45 or G80 without checking LAA and ACV.** G45 requires LAA Max 70%, ACV Max 40%. G80 requires LAA Max 50%, ACV Max 35%. These aggregate quality tests must be conducted.
4. **Confusing G-series (CBR-based) with GCS-series (aggregate quality-based).** G80 is a natural/blended granular material tested for CBR. GCS-D is a graded crushed stone tested for aggregate quality. They are different material types.

## M.7 Worked Example -- Kenya

**Scenario:** Specify pavement materials for a Kenya Class B road with traffic class TC3. The pavement structure from RDM 3.4 specifies: surfacing (DSD), base (G80 or GCS-D), sub-base (G45 or G30), improved subgrade (G14).

**Step 1: Base layer -- G80 or GCS-D**
- **G80 (Chart GM10):** CBR >= 80% at 95% MDD T180 and 4-day soak, PI Max 10, PM Max 250, LAA Max 50%, ACV Max 35%. Grading per Chart GM10 envelope.
- **GCS-D (Chart GM11):** Crushed stone Class D, CBR >= 80% 4-day soak, ACV Max 35%, LAA Max 50%. Partially weathered granites, basalts, and other rocks.
- Available crushed stone from quarry has ACV 28%, LAA 42%, CBR 95%: **Passes both G80 and GCS-D requirements.**

**Step 2: Sub-base layer -- G45 or G30**
- **G45 (Chart GM9):** CBR >= 45% at 95% MDD T180, PI Max 10, LAA Max 70%, ACV Max 40%. Grading per Chart GM9.
- Available lateritic gravel has CBR 52%, PI 8, LAA 55%: **Passes G45 requirements.**

**Step 3: Improved subgrade -- G14**
- **G14 (Chart GM4):** CBR >= 14% at 95% MDD T180, PI Max 30, LL Max 50, Swell Max 1.0%, PM Max 2500, Organic Matter Max 2%, Cu Min 6.
- Available clayey sand has CBR 18%, PI 22, LL 40: **Passes G14 requirements.**

**Step 4: Verify test standards**
- All CBR tests per AASHTO T 193 at 95% MDD (AASHTO T180), 4-day soak
- Atterberg limits per BS EN ISO 17892-12
- LAA per ASTM C131, ACV per BS 812-110
- Classification per ESCS (BS EN ISO 14688-2:2018)

**Output:** Materials specified and verified against RDM 3.3 Charts GM4, GM9, GM10/GM11.
