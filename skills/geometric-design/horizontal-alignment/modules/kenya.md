### M.1.2 Institutional Context

Same as Skill #13 Kenya module -- Kenya RDM is the mandatory standard for all classified roads in Kenya (KeNHA, KeRRA, KURA, County Governments). The RDM was developed by TRL Limited (UK) for MoRT with AfDB funding. Departures from standards require written approval from the Chief Engineer for Roads (Section 1.7).

### M.1.3 Standard Philosophy

Kenya RDM 1.3 provides the minimum radius formula R = V^2 / (127(e + f)) with tabulated results for emax values of 4%, 6%, and 8%. The normal maximum permissible super-elevation rate is **6%** (Section 5.9.1). In particular cases where major constraints exist, emax = 7% may be applied, and on loop ramps at interchanges emax up to 8% may be adopted with approval of the Chief Engineer Roads. In urban areas, an upper limit of 4% should be used except on high-speed urban roads where 6% is acceptable. This is a "shall" standard -- more prescriptive than the Uganda MoWT approach.

### M.2.2 Country-Specific Formulas

**Minimum radius (Kenya RDM -- same as AASHTO):**

```
Rmin = V^2 / (127 x (e + f))
```

Where V is in km/h, e is in decimal (e.g., 0.06), f from Table 3.14 (side friction factors).

**Example from the manual (verified):**
- V = 80 km/h, emax = 6% (0.06), f = 0.13
- Rmin = 80^2 / (127 x (0.06 + 0.13)) = 6400 / (127 x 0.19) = 6400 / 24.13 = **265 m** (matches Table 5.1)

**Clothoid transition parameter:**

```
A_min = 0.21 x V_D^1.5
```

Where V_D is design speed in km/h (Equation 5.4, RDM 1.3).

**Super-elevation runoff length:**

```
Lo_min = ((e - e_o) / delta_S_max) x (w / 2)
```

Where e = super-elevation on curve, e_o = crossfall on straight, w = carriageway width (Section 5.9.4).

**Absolute minimum horizontal curve radius = 15 m** (Section 5.8). This applies to all design standards. Hairpin curves with radius 20 m or less require departures from standards (Section 5.12).

### M.2.3 Superelevation Application

Kenya RDM 1.3 provides superelevation rate tables for emax = 4% (Table 5.9), emax = 6% (Table 5.10), and emax = 8% (Table 5.11). These tables give the superelevation rate for each combination of radius and design speed.

Normal crossfall is **2.5%** for paved roads and **4%** for gravel roads (Section 4.3.4). Earth roads use 5% crossfall.

Superelevation development has two components: tangent runout (from normal camber to zero crossfall) and super-elevation runoff (from zero crossfall to full superelevation). Current practice places approximately two-thirds of the runoff on the tangent approach and one-third on the curve (Section 5.10).

## M.3 Country-Specific Reasoning

### M.3.1 Decision Logic (Kenya Branch)

```
Step 2 (Kenya): Maximum superelevation
  +-- Rural (standard) -> emax = 6%
  +-- Rural (constrained, with Chief Engineer Roads approval) -> emax = 7%
  +-- Loop ramps at interchanges (with approval) -> emax = 8%
  +-- Urban (standard) -> emax = 4%
  +-- High-speed urban -> emax = 6%
  +-- Expressway (DR1) -> emax = 6%, max gradient 4%
  \-- Check ToR for project-specific requirements

Step 3 (Kenya): Minimum radius
  +-- Paved roads: Look up from Table KH.1 using design speed and emax
  +-- Unpaved roads: Look up from Table KH.2 (emax = 4%)
  +-- Cross-check: R >= 15 m absolute minimum (Section 5.8)
  \-- If expressway: use DR1 standards from Chapter 14

Step 4 (Kenya): Transition curves
  +-- Required if R < threshold from Table KH.7
  +-- Not required for low design speeds or low classification roads
  +-- Clothoid parameter A_min from Equation 5.4
  \-- Compound curves: R1 should not exceed 1.5 x R2 (Section 5.11)
```

### M.3.2 Professional Judgment (Kenya-Specific)

**Kenya uses emax = 6% as the standard, not 7% or 8%.** This is a critical difference from Uganda (emax = 7%). Super-elevation exceeding 6% is rarely used in current Kenya practice and requires Chief Engineer Roads approval. Designers should use emax = 6% unless the project has exceptional constraints.

**Motorcycle (boda-boda) traffic is significant on Kenya roads.** The RDM specifically addresses motorcycle considerations (Section 3.6) including motorcycle-friendly safety barriers, adequate clear zones, and surface consistency at superelevation transitions. This is particularly relevant for Class D and E roads where motorcycle traffic may exceed motorised vehicle traffic.

**The compound curve ratio of 1.5:1 is stricter than some international standards.** Kenya RDM requires R1 not to exceed 1.5 x R2 for compound curves (Section 5.11). The ratio of consecutive curve radii should fall within the "very good" or "good" zone of Figure 5.3.

### M.3.3 Common Errors (Kenya-Specific)

1. **Using emax = 7% or 8% without Chief Engineer Roads approval.** The Kenya standard is 6% for rural roads. Higher values require formal departures process approval.
2. **Confusing Kenya side friction factors with Uganda MoWT values.** Kenya uses different friction values from Table 3.14 (e.g., f = 0.13 at 80 km/h vs f = 0.14 in Uganda at 80 km/h). Always use the Kenya table.
3. **Not checking the straight length limits.** Kenya specifies both minimum (6V) and maximum (20V) straight lengths (Table 5.4). Long straights encourage speeding; short straights create broken-back curves.
4. **Ignoring the transition curve threshold.** For design speeds of 70 km/h and above, transition curves are required for radii below the Table 5.8 values. This is mandatory, not advisory.

### M.5.2 Standards Compliance Checks

- **Check:** Curve radius >= minimum from Table KH.1 for the selected design speed and emax
- **Standard:** RDM 1.3, Table 5.1
- **Pass condition:** R >= tabulated minimum radius
- **Fail action:** SENIOR REVIEW -- Radius below minimum for design speed. Requires Departure from Standards approval from Chief Engineer Roads per Section 1.7.

- **Check:** Superelevation does not exceed emax for the context
- **Standard:** RDM 1.3, Section 5.9.1
- **Pass condition:** e <= 6% (rural) or e <= 4% (urban)
- **Fail action:** Flag -- requires Chief Engineer Roads approval for higher emax.

### M.5.3 Departures / Relaxations Process

Kenya RDM has a **formal departures process** (Section 1.7). Where the designer departs from a standard, written approval shall be obtained from the Chief Engineer for Roads. The submission shall include: road identification, the parameter being departed from, standard value and proposed departure value, reason, safety mitigation, and justification. This is significantly more prescriptive than Uganda MoWT's informal approach.

## M.7 Worked Example -- Kenya

### Example: Class B National Highway, Rolling Terrain

**Given:**
- Road: Class B National Highway (KeNHA)
- Design Standard: DR2 (AADT > 4000)
- Terrain: Rolling
- Design Speed: 90 km/h (from Table K.4, Class B, Rolling: max 90, min 85)
- emax: 6% (rural standard)
- Proposed curve radius: 400 m

**Solution:**

**Step 1 -- Check minimum radius:**
From Table KH.1: At 90 km/h with emax = 6%, Rmin = 355 m
Proposed 400 m > 355 m -> Compliant

**Step 2 -- Check against desirable radius:**
From Table KH.3: Desirable R > 3780 m for rural arterials. R = 400 m is well below desirable but above the minimum. Acceptable but not ideal.

**Step 3 -- Calculate superelevation:**
Using Rmin = V^2 / (127 x (e + f)):
- 400 = 90^2 / (127 x (e + 0.12))
- 400 = 8100 / (127 x (e + 0.12))
- 127 x (e + 0.12) = 8100 / 400 = 20.25
- e + 0.12 = 20.25 / 127 = 0.1594
- e = 0.1594 - 0.12 = 0.0394 -> **e = 3.9%**

Cross-check with Table 5.10 (emax = 6%): At R = 400 m and 90 km/h, e = 3.9%. Consistent.

**Step 4 -- Transition curve:**
From Table KH.7: At 90 km/h, transition required if R < 480 m.
R = 400 m < 480 m -> **Transition curve required**

Minimum clothoid parameter: A_min = 0.21 x 90^1.5 = 0.21 x 854.6 = **179 m**
Minimum transition length from run-off: L_o = 50 m (Table KH.6 at 90 km/h)

**Step 5 -- Curve widening:**
From Table 4.15: At R = 400 m (> 250 m), no curve widening required for single lane. For two-lane section, widening = 0 m (R > 250 m). No widening needed.

**Result:**

| Parameter | Value |
|-----------|-------|
| Minimum radius (90 km/h, emax 6%) | 355 m |
| Proposed radius | 400 m |
| Compliance | Above minimum |
| Required superelevation | 3.9% |
| Transition curve | Required (R < 480 m) |
| Clothoid parameter A_min | 179 m |
| Curve widening | None (R > 250 m) |
